## Campaign goal
Build early awareness, create repeat exposure for the new crypto line, and use WC listing media to generate steady daily content that drives traffic to the Etsy shop.

## Core strategy
1. **Crypto range** = simple, recognisable, easy-to-share logo-led posts.
2. **WC range** = motion-first content using listing videos for reach and engagement.
3. **Daily posting cadence** = 1 main post per day, adapted per platform.
4. **Content mix** = 60% product showcase, 20% lifestyle/brand, 20% direct conversion.

## Platform use
### Instagram
- Primary visual channel.
- Use carousel posts for product images.
- Use Reels for WC videos.
- Stories can repost the daily main content.

### TikTok
- Best home for WC videos.
- Short edits, quick hooks, product close-ups, loop-friendly clips.

### Facebook
- Use for straightforward product launches, shop links, and cross-posted Reels/video.

### X / Twitter
- Use for short launch posts, product threads, crypto-angle commentary, and repeat reminders.

## Weekly structure
- **Monday:** new or featured product
- **Tuesday:** product close-up / carousel
- **Wednesday:** video feature
- **Thursday:** themed collection post
- **Friday:** lifestyle / why-this-design post
- **Saturday:** soft-sell weekend push
- **Sunday:** roundup / best-of / shop CTA
