# Brand sub-sites

This directory contains static sub-site pages for the Slakka initiative brands.

Current brands:
- kapp
- beet
- klab
- moov
- kask
- lask
- pai
- kovr
- korr
- simpl

Each brand has:
- `index.html`
- `products.html`
- `about.html`

These are deployment-ready static pages prepared for subdomain mapping such as `kapp.slakka.com`.
